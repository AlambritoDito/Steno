# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-03-05

### Added

- Real-time audio transcription using Whisper Large V3 Turbo via MLX
- Markdown note-taking with CodeMirror 6 editor
- Image support via drag & drop (base64-embedded in Markdown)
- Session management: create, save, load, export, and delete
- WebSocket-based real-time transcript streaming
- Multilingual UI: English and Spanish with runtime switching
- Auto-detection of browser language
- Microphone selection and audio level monitoring
- Session export as `.md` files
- Dark mode UI
- Full test suite (42 tests)
